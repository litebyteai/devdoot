# Devdoot TypeScript Example

This is a TypeScript project demonstrating how to use `devdoot` with strict type checking and `tsconfig.json`.

---

## 🚀 Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the basic tracing and exception safety demo:
   ```bash
   npm run demo
   ```

3. Run the Express HTTP middleware tracing server:
   ```bash
   npm run express
   ```

4. Run the Axios HTTP header context propagation demo:
   ```bash
   npm run axios
   ```
