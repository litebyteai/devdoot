# Devdoot Node.js Example

This is a pure JavaScript Node.js project demonstrating how to use `devdoot` using modern ES Module syntax (`import`/`export`).

---

## 🚀 Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the basic tracing and exception safety demo:
   ```bash
   npm run demo
   ```

3. Run the Express HTTP middleware tracing server:
   ```bash
   npm run express
   ```
