# Devdoot Examples

This directory contains two separate, fully-configured sub-projects demonstrating how to integrate `devdoot` from the live NPM registry:

1. **[node-js](file:///d:/Projects/nodejs/automation/devdoot/uses-example/node-js)**: Pure JavaScript examples using standard ES Modules and plain Node.js.
2. **[typescript](file:///d:/Projects/nodejs/automation/devdoot/uses-example/typescript)**: TypeScript examples utilizing compiler options, type checking, and standard TypeScript features.

Navigate to either folder to get started!
